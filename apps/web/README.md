# Grapepay Web (Next.js Application) 🌐

This is the frontend layer of the Grapepay Command OS, built with Next.js 15 for high-speed performance and professional-grade interactivity.

## 🚀 Key Features

- **14+ Operational Routes**: Comprehensive coverage of the Quick Commerce lifecycle.
- **Dynamic Simulations**: Built-in real-time data randomizers to demonstrate operational flow.
- **Modular Architecture**: Reusable UI components (Cards, Tickers, Pipelines).
- **Vercel Ready**: Optimized for edge deployment with zero configuration.

## 🛠 Tech Stack

- **Framework**: Next.js 15
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Styling**: CSS Modules with CSS Variables (Themeable)

## 📦 Local Development

```bash
npm install
npm run dev
```

## 🚢 Deployment

The easiest way to deploy is via [Vercel](https://vercel.com):

1. Connect your Github repository.
2. Select `apps/web` as the root directory.
3. Deploy.
